🔧 Canvas API Token Setup

To get your Canvas API token:

  1. Log into your Canvas account
  2. Go to Account → Settings
  3. Scroll to Approved Integrations
  4. Click + New Access Token
  5. Set purpose: "Student Success Prediction"
  6. Copy the generated token
Security: Tokens are stored locally in your browser only.

🔧 PowerSchool API Setup

To set up PowerSchool API access:

  1. Contact your PowerSchool district administrator
  2. Request API access for "Student Success Prediction"
  3. Provide these required scopes: students, schools, courses, grades
  4. Administrator creates OAuth2 client credentials
  5. Receive Client ID and Client Secret
Note: Requires district administrator approval and API licensing.

🔧 Google Classroom API Setup

Follow these steps to enable Google Classroom integration:

  1. Go to Google Cloud Console
  2. Create a new project or select existing project
  3. Enable the Google Classroom API
  4. Create OAuth2 credentials (Desktop application type)
  5. Download the credentials JSON file
  6. Place it as config/google_credentials.json
  7. Restart the application
📝 Need detailed instructions? See docs/Google_Classroom_Setup.md
