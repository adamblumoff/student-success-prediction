PowerSchool SIS Integration

Your district's PowerSchool URL

How to get PowerSchool API Credentials:

  1. Log in to PowerSchool Admin Portal
  2. Go to System ‚Üí System Settings ‚Üí Plugin Management Configuration
  3. Click Enable API if not already enabled
  4. Create a new API Client:
    • Name: "Student Success Prediction"
    • Description: "K-12 student risk analysis"
    • Grant Type: Client Credentials
  5. Set required API Scopes:
    • Students: Read
    • Schools: Read
    • Grades: Read
    • Attendance: Read
    • Discipline: Read
  6. Copy the Client ID and Client Secret

Note: You need PowerSchool Administrator access to create API clients.
